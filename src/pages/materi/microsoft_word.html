A. Pengertian Microsoft Office

    Microsoft office adalah software atau perangkat lunak yang berisi paket aplikasi perkantoran yang dibuat oleh perusahan yang sangat terkenal yaitu Microsoft. Salah satu aplikasi yang terdapat dalam program Microsoft office adalah Microsoft Word

B. Pengertian Microsoft Word

    Microsoft Word merupakan sebuah aplikasi program pengolah kata (word processor) yang dibuat oleh Microsoft Corporation. Program ini biasanya digunakan para pemakai komputer untuk kegiatan tulis-menulis. Seperti pembuatan surat, proposal, artikel, brosur, booklet, dan lain-lain dengan aneka font (huruf) dan layout yang tersedia.

C. Sejarah Microsoft Word

    Dari awal berdirinya Microsoft word, Pengembangan Multi-Tool Word, Multi-Tool Word Berganti Nama Menjadi Microsoft Word, Perkembangan Microsoft Word, Word For Windows, Word For Mac , Microsoft Word 2010, Microsoft Word 2013, Microsoft Word 2019, Microsoft Word terbaru yaitu Microsoft word 2021.

D. Fungsi Microsoft Word

    Dari awal berdirinya Microsoft word, Pengembangan Multi-Tool Word, Multi-Tool Word Berganti Nama Menjadi Microsoft Word, Perkembangan Microsoft Word, Word For Windows, Word For Mac , Microsoft Word 2010, Microsoft Word 2013, Microsoft Word 2019, Microsoft Word terbaru yaitu Microsoft word 2021.

  • 1. Membuat dokumen – Dengan program ini pengguna dapat membuat dokumen dalam bentuk apa pun, seperti halnya teks, gambar, grafik ataupun tabel. Dan dokumen ini biasanya berupa surat, laporan, jurnal, buku, serta dokumen teks lainnya.
  • 2. Mengedit dokumen – Apabila terjadi sebuah kesalahan dalam pengetikan, pengguna dapat mengubah isi dari dokumen tersebut.
  • 3. Alat keluaran dokumen – Hasil akhirnya tentu saja dapat disimpan dalam bentuk softcopy maupun hardcopy.
