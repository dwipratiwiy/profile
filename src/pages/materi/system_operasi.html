A. Pengertian System Operasi

    Sistem operasi merupakan sebuah penghubung antara pengguna komputer dengan perangkat keras komputer.

B. Perkembangan System Operasi

    Sistem operasi mengalami perkembangan yang sangat pesat yang dapat dibagi ke dalam empat generasi (Tanenbaum, 2005) :

  • 1. Generasi Pertama (1945-1955). Pertama merupakan awal perkembangan sistem komputasi elektronik sebagai pengganti sistem komputasi mekanik.
  • 2. Generasi Kedua (1955-1965). Generasi kedua memperkenalkan Batch Processing Sistem, yaitu dengan menjalankan beberapa tugas sekaligus, job yang dikerjakan dalam satu rangkaian kemudian dieksekusi secara berurutan dan sudah dapat melakukan dengan lebih baik pada resource computer.
  • 3. Generasi Ketiga (1965-1980). Pada generasi ini perkembangan sistem operasi dikembangkan untuk melayani banyak pemakai sekaligus.
  • 4. Generasi Keempat (Pasca 1980an). Dewasa ini, sistem operasi dipergunakan untuk jaringan komputer. Pemakai menyadari keberadaan komputer-komputer yang saling terhubung satu sama lainnya.

C. Perkembangan System Operasi

    Microsoft Windows atau sering disebut dengan Windows merupakan sistem operasi yang dibuat dan dikembangkan oleh Microsoft Corporation, perusahaan perangkat lunak yang didirikan oleh Bill Gates pada tahun 1975 dan berkantor pusat di Redmond, Washington, Amerika Serikat.

D. Sejarah system operasi windows

    Dimulai dari windows 1.0, windows 2.0, windows 2.1, windows 3.0,windows NT 3.1,windows 95, windows 98, windows 2000, windows ME, windows XP, windows server 2003, windows vista, windows home server, windows server, windows 7, windows 8, windows 10 sampai sekarang windows 11.

E. Kelebihan Dan Kekurangan System Operasi Windows

    Kelebihannya Pengguna bisa membuka lebih dari satu file dalam waktu yang bersamaan. Pengguna bisa sharing dari data aplikasi yang lain dengan mudah. Kekurangannya adalah Sistem security yang lemah dan mudah dijebol oleh para hackers.
